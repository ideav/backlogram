## 5. Трекинг конверсий (в коде уже сделано)

Счётчик Метрики **108758829** встроен в `index.html`. Вызов цели `lead`
добавлен во все формы-заявки (`src/lib/metrika.ts` → `reachGoal`, PR #429).
При успешной отправке шлётся `ym(108758829, 'reachGoal', 'lead', { source })`.

Остался **один шаг на стороне Метрики** (после деплоя PR #429):
1. В Яндекс.Метрике (счётчик 108758829) создать цель типа **«JavaScript-событие»** с идентификатором `lead`.
2. В кампании выбрать эту цель ключевой для «Максимум конверсий».

До создания цели «Максимум конверсий» переключать рано — на старте «Максимум кликов».

## 6. Порядок запуска

1. Дождаться мержа PR #429 и деплоя — проверить, что `https://ideav.ru/konstruktor-prilozhenij.html` открывается (200) и форма отправляется.
2. (желательно) Настроить цель в Метрике — §5.
3. Создать кампанию с параметрами §1, добавить 3 объявления §3 в одну группу, ключи §4, минус-слова §4b.
4. Стратегия «Максимум кликов», недельный бюджет ≈3 500 ₽, ставку/цену клика оставить авто.
5. Через 2 недели / после 10 конверсий — переключить на «Максимум конверсий» по цели `lead`.
6. Раз в неделю: чистить поисковые запросы (добавлять новые минус-слова), сравнивать 3 объявления по CTR/конверсии, отключать слабое.
