# Смежные и косвенные ключи (issue #430)

Идея: человек, который ищет «эффективная загрузка станков», планирует это в Excel
или подобном — то есть это наша ЦА, просто зашедшая с задачи, а не со слова
«конструктор». Бьём в **задачу и боль**, а в решении показываем замену Excel.

Эти группы можно использовать как **отдельные группы объявлений** в поиске и как
аудитории для **ретаргетинга**. Целевые страницы: пока — `/konstruktor-prilozhenij.html`;
после выката тематических лендингов (issue #431) — вести на профильную страницу
(напр. производственное планирование → страница про план производства).

## Объявления для производственной группы (проверены под лимиты Директа)

**На боль «загрузка станков»**
- Заголовок 1: `Устали от Excel для загрузки станков?`
- Заголовок 2: `Автоматизируйте планирование`
- Текст: `Интеграм: замените Excel, планируйте загрузку станков и стройте отчёты без кода.`

**На задачу «диспетчирование»**
- Заголовок 1: `Планирование производства без Excel`
- Заголовок 2: `Конструктор без программистов`
- Текст: `Диспетчирование, загрузка оборудования, отчёты по цеху. Настройка за 1 день.`

**На боль «учёт»**
- Заголовок 1: `Учёт станков вместо десятка таблиц`
- Заголовок 2: `Оперативный учёт в цехе`
- Текст: `Единая база вместо разъезжающихся Excel: выработка, загрузка, дашборды. Демо.`

## Как применять

1. **Поиск.** Отдельная группа под каждую боль (не мешать с «конструктор»/«no-code»):
   так заголовок бьёт точно в запрос, а CTR и Показатель качества выше.
2. **Ретаргетинг.** Кто искал «как автоматизировать диспетчирование», но не оставил
   заявку → догоняем баннером «Не нашли решение? Интеграм для планирования».
3. **Минус-слова** — общие из основного пакета (`yandex-direct-excel-constructor.md`, §4b);
   для производства **не** минусовать `1с`, `mes`, `erp`, `учёт`.
4. **Масштабирование на другие ниши.** Тот же приём (боль-задача → замена Excel)
   разворачивается на 10 тематик из issue #431 — там для каждой ниши свой лендинг и
   своя группа смежных ключей (склад, закупки, кадры, финансы, CRM, документооборот и т.д.).
