## Какой файл импортировать

- **`direct_712841876_filled.xlsx`** — **РЕАЛЬНАЯ кампания 712841876** (выгрузка из
  Директ Коммандера с настоящими ID групп/фраз/объявлений). Импортировать **этот**
  файл — повторная загрузка **обновит** кампанию, а не создаст дубли. Тип
  объявлений — «Комбинаторное», заголовки лежат в колонке «Комбинаторика».
  Сюда дозаполнены быстрые ссылки, уточнения и отображаемая ссылка.
- **`direct_campaigns_11_pages.xlsx`** — исходный шаблон **без реальных ID**
  (для первичного создания структуры). Повторный импорт этого файла **создаёт
  дубли** — использовать только один раз.

> Правило: файл без настоящих ID при каждом импорте создаёт новые объекты.
> Чтобы обновлять кампанию без дублей — правьте в Коммандере или импортируйте
> выгрузку с ID (`direct_712841876_filled.xlsx`).

Файл **`direct_campaigns_11_pages.xlsx`** — заполненный официальный шаблон
Директа (лист «Тексты» в формате Директ Коммандера), сохранены листы
«Регионы» и «Словарь значений полей».

## Быстрые ссылки, уточнения, UTM — заполнены в файле

Всё это уже проставлено в XLSX, вручную в кампании ничего добивать не нужно:

- **UTM** — колонка «Ссылка» у каждой строки:
  `utm_source=yandex&utm_medium=cpc&utm_campaign=<slug>&utm_content={ad_id}&utm_term={keyword}`
  (`{ad_id}`/`{keyword}` — подстановки Директа).
- **Быстрые ссылки** (4 шт., одинаковые во всех группах, адрес — под каждую страницу):

  | Заголовок | Описание | Адрес |
  | :--- | :--- | :--- |
  | Заказать демо | Настроим под вашу задачу за 1 день | `/<страница>.html#zayavka` |
  | Как это работает | Файл, настройка, готовое приложение | `/<страница>.html` |
  | Готовые решения | 10 готовых решений по задачам | `/resheniya.html` |
  | База знаний | Статьи и разборы по автоматизации | `/knowledge-base.html` |

  У адресов быстрых ссылок свой UTM (`utm_content=sitelink`).
- **Уточнения** (общие): Российская платформа · On-premise · Без программистов ·
  Демо за 1 день.
- **Отображаемая ссылка** — короткое слово под страницу (konstruktor, sklad, crm…).

Лимиты соблюдены: заголовок БС ≤30, описание БС ≤60, уточнение ≤25,
отображаемая ссылка ≤20 символов.

## Важно про «кампании» vs «группы»

Лист «Тексты» в этом формате Директа не содержит колонки названия кампании —
он описывает **группы одной кампании**. Поэтому 11 страниц заполнены как
**11 групп** (Номер группы 1–11). Варианты запуска:
- **Одна кампания, 11 групп** — импортировать файл как есть в одну кампанию.
- **11 отдельных кампаний** — импортировать в 11 кампаний (или скажите — соберу
  11 отдельных файлов, по одному на кампанию).

## Лимиты и заметки

- Заголовок 1 ≤ 56, Текст ≤ 81 — соблюдены везде.
- Заголовок 2: у 10 ниш ≤ 30; у группы «Конструктор вместо Excel» объявления
  взяты **из вашего примера** (`direct_example.xlsx`) как есть — там Заголовок 2
  40/33/34 символа (Директ подрежет при импорте, если потребуется).
- UTM: `utm_source=yandex&utm_medium=cpc&utm_campaign=<slug>&utm_content={ad_id}&utm_term={keyword}`
  — привязано к цели Метрики `lead` (счётчик 108758829, см. формы страниц).
- Минус-слова — общие для всех групп; при желании чистить по нишам отдельно.
- Смежные/косвенные ключи (производство и т.п.) — в `keywords-adjacent-430.md`.
